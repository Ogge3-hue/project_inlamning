#Project using React + Vite

Have been using react framwork, and Vite as building tool and dev-server to develop a e-shop with dummyjson api.
React tools used:
useState
useEffect
useMemo
useContext

Packages used developing this is:
React-Router-Dom
Bootstrap
React-Bootstrap

I have been using Routes to navigate the Hero section, and links and Nav-links to move around the site
And the changes made is saved in the local-storage. 
Cart Context got global scope.
Also added a debounce on the Store page search will wait until typing stopped to do the fetch from API.
